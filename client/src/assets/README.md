# Assets Folder

This folder contains static assets for the AK Mobiles application.

## Logo
Place your logo image files here:
- logo.png (primary logo)
- logo.svg (vector logo)
- favicon.ico (favicon)

## Usage
Import assets in components like this:
```javascript
import logo from '../assets/logo.png';

// Then use in JSX:
<img src={logo} alt="AK Mobiles" className="h-8 w-auto" />
```
